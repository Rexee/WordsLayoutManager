# WordsLayoutManager
Custom implementation of RecyclerView.LayoutManager based on LinearLayoutManager
![main][Mainimg]

Usage:
```java
RecyclerView recycler = (RecyclerView) findViewById(R.id.recycler);
recycler.setLayoutManager(new WordsLayoutManager(this));
```
